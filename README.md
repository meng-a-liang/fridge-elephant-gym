# 把大象放进冰箱 - OpenAI Gym强化学习环境

## 项目简介
本项目基于OpenAI Gym规范，实现了“把大象放进冰箱”的强化学习训练环境，核心目标是为毕业论文提供可拓展、模块化的训练框架。

## 核心架构（可拓展性设计）
| 模块                | 作用                                  | 拓展方向                                  |
|---------------------|---------------------------------------|-------------------------------------------|
| `fridge_gym/elements/` | 封装游戏元素（冰箱、大象）| 新增元素（比如门、场景）、加动画/物理属性  |
| `fridge_gym/envs/`     | 定义Gym标准环境                       | 新增难度版本（随机位置、多大象/冰箱）|
| `fridge_gym/utils/`    | 渲染、工具函数                        | 新增日志、数据记录、可视化分析工具        |
| `examples/`            | 演示脚本                              | 新增强化学习训练脚本（DQN/PPO）|

## 核心状态向量
状态向量维度：`[冰箱开关状态(0/1), 冰箱x坐标, 冰箱y坐标, 大象x坐标, 大象y坐标]`
- 维度0：冰箱开关（0=关闭，1=打开）
- 维度1-2：冰箱中心坐标（适配窗口缩放）
- 维度3-4：大象中心坐标（适配窗口缩放）
